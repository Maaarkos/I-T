# 🐙 GIT: Basic Concepts

*   **GIT** -> is a free technology (a program) used for tracking changes in files and source code.
*   **GitHub** -> is a company (owned by Microsoft) that created a web portal based on the GIT technology.
*   **GitLab** -> is a different, independent company that created its own competing web portal, also based on the GIT technology.
